## 🚀 INSTALACIÓN RÁPIDA (4 PASOS)

### 1️⃣ Ejecutar SQL
```bash
# Doble clic en:
instalar_actividades.bat

# O manualmente:
psql -U postgres -d Gestiondetareas -f crear_tabla_actividades.sql
```

### 2️⃣ Compilar
```bash
mvn clean package
```

### 3️⃣ Desplegar
```
Copia target/SistemaGestionTareas.war a Tomcat/webapps/
Reinicia Tomcat
```

### 4️⃣ Probar
```
1. Accede: http://localhost:8080/SistemaGestionTareas
2. Click en "📁 Mis Actividades"
3. Crea tu primera actividad
4. Agrega tareas a la actividad
5. Agrega subtareas a las tareas
```

## 📊 ESTRUCTURA DE BASE DE DATOS

```sql
-- Nueva tabla
actividades (
    id, usuario_id, titulo, descripcion,
    fecha_inicio, fecha_fin, prioridad, estado,
    color, fecha_creacion
)

-- Tabla modificada
tareas (
    ...(campos existentes)...,
    actividad_id  -- NUEVO CAMPO (FK → actividades)
)

-- Tabla existente (sin cambios)
subtareas (
    id, tarea_id, titulo, descripcion,
    completada, fecha_creacion
)
```

**Relaciones:**
- 1 Actividad → N Tareas
- 1 Tarea → N Subtareas
- 1 Tarea → 0..1 Actividad (opcional)
