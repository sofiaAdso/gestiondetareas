    ❓ "Puerto 8080 ya está en uso"
       → Ejecuta: netstat -ano | findstr :8080
       → Mata el proceso o cambia el puerto

    ❓ "Java no encontrado"
       → Instala Java 17 JDK
       → Verifica con: java -version

    ❓ "Error de compilación Maven"
       → Ejecuta: .\mvnw.cmd clean install

    ❓ "Página en blanco"
       → Verifica que PostgreSQL esté corriendo
       → Revisa credenciales de BD
       → Mira los logs en la consola
